#### Quiz1

编写一个C++程序，要求用户输入一个以long为单位的距离，然后将它转化为码（1long=220码）

```cpp
#include<iostream>

const double Factor = 220.0;

int main(){
    double dist = 0.0;
    std::cout<<"Enter a distance (long): ";
    std::cin>>dist;
    std::cout<<dist<<" long = "<<dist * Factor<<" yd\n";
    return 0;
}
```
